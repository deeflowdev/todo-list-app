# 📝 Todo List App

A responsive Todo List application built with React and Tailwind CSS. Users can add, complete, delete tasks, switch between multiple themes, and persist data using Local Storage.

## Features

- Add and delete tasks
- Mark tasks as completed
- Multiple themes (lavender, sage, rose, midnight)
- Persistent storage with Local Storage
- Responsive UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Remix Icons

## What I Practised

- React hooks (`useState`, `useEffect`, `useRef`)
- Component-based architecture
- Conditional rendering
- Dynamic styling with themes
- Array methods (`map`, `filter`)
- Local Storage integration
